# Travel Companion App

## Overview
The Travel Companion App is an Android application developed using Android Studio and Java. The purpose of the application is to help users perform useful travel-related conversions such as currency conversion, fuel/distance conversion, and temperature conversion.

The application provides a simple and user-friendly interface where users can enter values, select conversion categories, and instantly view the converted results.

---

# Features

## Currency Conversion
- Convert between different currencies
- Easy input and result display

## Fuel and Distance Conversion
- Convert fuel consumption values
- Convert distance measurements

## Temperature Conversion
- Convert Celsius to Fahrenheit
- Convert Fahrenheit to Celsius

## User Interface
- Simple and clean design
- Spinner dropdown menus
- Input validation
- Result display area

## Validation and Error Handling
- Prevents empty input fields
- Handles invalid user input
- Displays helpful messages to users
